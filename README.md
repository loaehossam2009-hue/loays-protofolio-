# Personal Portfolio Website

## Project Overview

This project is a personal portfolio website developed as part of the Web Development and Programming course.  
The website showcases personal information, skills, projects, and contact details.

The main goal of this project is to demonstrate front-end development skills using modern web technologies.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

---

## Website Features

- Responsive design for desktop, tablet, and mobile
- Navigation bar for easy page navigation
- Dark / Light mode toggle
- Projects gallery section
- Contact form
- Multi-page website structure

---

## Website Pages

### Home
The homepage introduces the portfolio owner and includes a short description and navigation buttons.

### About
This page contains personal information, background, and education details.

### Projects
A gallery showcasing different web development projects with descriptions.

### Contact
A contact form where users can send messages and find contact information.

1. Download the project files.
2. Open the project folder.
3. Run the file `index.html` in any web browser.
